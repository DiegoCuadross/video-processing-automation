# Video Processing Automation

Script en Windows (.bat) para:

- Recorrer carpetas de una filmadora
- Detectar archivos .MOD
- Convertirlos a .mp4 usando FFmpeg
- Mantener estructura de carpetas
- Evitar reprocesamiento
- Detectar archivos corruptos

## Requisitos

- FFmpeg instalado

## Uso

Editar rutas en el script:

ORIGEN=F:\SD_VIDEO  
DESTINO=D:\Filmadora videos

Ejecutar:

procesar_videos.bat
